# 6-Clip Starter Recording Checklist

Use this during the first room session before scaling to the full micro-pilot.

## Clip Pass Rule

A clip passes only if all are true:

- clean vocal only
- no long silence at start or end
- no clipping
- no room-noise spike
- one intelligible phrase
- alignment looks plausible on listen-check

If a clip fails, retake it immediately.

## Session Checks

- room is quiet
- mic position is stable
- input gain is fixed
- recording chain has no reverb, EQ, or compression
- files are mono WAV
- each take is short and isolated
